# Демо приложение для резюме
Посмотреть демо — https://projects.pure-code.ru

## Описание проекта
Kanban доска для hr-менеджеров. \
\
Из особенностей — механизм drag and drop реализован с нуля.

Основные инструменты:
- React
- Typescript
- MobX
- Styled Components
